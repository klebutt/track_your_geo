## Citation behavior (expected quirks)

**Perplexity** is working well — avg **8.4 domains** per probe (e.g. `theinfatuation.com`, `tripadvisor.com`, `reddit.com`).

**OpenAI** mostly cites `google.com` because search annotations point at Google Maps URLs. That is the existing OpenAI path, not a regression.

**Gemini** returns `vertexaisearch.cloud.google.com` — LiteLLM's grounding metadata uses Google redirect URLs rather than final source domains. Citations are being captured, but domain quality is weaker than Perplexity. Improving that would be a follow-up extraction tweak, not a deploy issue.
