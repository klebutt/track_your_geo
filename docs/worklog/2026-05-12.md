### GEO methodology (neutral probes)

- **Removed all brand-in-prompt templates** (`{brand}` in the question text), which inflated visibility because the model was steered to repeat the brand name.
- **Expanded each scenario to 10 brand-neutral probes** so visibility is sampled more often per run (higher API cost, smoother signal).
- Tuned one overly niche prompt (“Bombay-style cafe”) toward a broader Indian-restaurant question to reduce implicit steering toward one chain.

### Scoring realism (documentation)

- Added **`docs/geo-scoring-realism.md`**: gap vs consumer ChatGPT-style search, options to improve fidelity (consumer panel, search-grounded APIs, intent-rich probes, beyond-substring metrics, multi-metric reporting, calibration), and honest limits of the current index.
